There are several serial interfaces in hardware communacation.

Such as UART, I2C, SPI, CAN, LIN, ..... etc.

I will tell you some serial interfaces.

UART

This communication uses two pins; 'TX' and 'RX'.

This can do an error check using a 'Parity' bit.

I2C

This communication uses two pins; 'SCL' and 'SDA'.

This was developed by 'Philips' corporation.

SPI

This communication uses four pins; 'SCLK', 'MOSI', 'MISO' and 'SS'.

This was developed by 'Motorola' corporation.
