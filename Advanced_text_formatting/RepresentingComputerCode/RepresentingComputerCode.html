CC = gcc
			TARGET = HelloWorld.out
			OBJS = HelloWorld.o
			SRCS = HelloWorld.c

			$(TARGET) : $(OBJS)
				$(CC) -o $@ $(OBJS)

			$(OBJS) : $(SRCS)
				$(CC) -c -o $(OBJS) $(SRCS)

"CC" means the used compiler.

"TARGET" means the executable file.

"OBJS" means the objective file.

"SRCS" means the source file.

If you want to copy all above source code above Makefile,

push buttons in your keyboard Ctrl C.

And to want to paste it, do push Ctrl V.

Anyway when you execute the Makefile, following sentences will appear in your monitor.


			gcc -c HelloWorld.o HelloWorld.c
			gcc -o HelloWorld.out HelloWorld.o
